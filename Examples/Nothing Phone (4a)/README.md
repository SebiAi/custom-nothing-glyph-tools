# Nothing Phone (4a)
We differentiate between two modes on the Nothing Phone (4a):
| Mode | Description |
| - | - |
| **6 Zones** | Uses all [\[Glyphs\]](../../docs/1_Terminology.md#glyphs) excluding the red camera recording one. |
| **7 Zones** | Uses all [\[Glyphs\]](../../docs/1_Terminology.md#glyphs) including the red camera recording one - see caution below! |

> [!CAUTION]
> The *Nothing Phone (4a)* **7 Zones** mode (addressing the red camera indicator [\[Glyph\]](./1_Terminology.md#glyphs)) is not officially supported by Nothing and may break unexpectedly in the future!
